Nothing fancy at the moment, just a personal dotfiles repo.

Install instructions
====================

1. Install oh-my-zsh
2. clone this repo to ~/.dotfiles directory
3. create a symlink for the candykb.zsh-theme to the oh-my-zsh themes directory
4. create the following symlinks
  * ~/dotfiles/vim/ symlink to ~/.vim/
  * ~/dotfiles/vimrc symlink to ~/.vimrc
  * ~/dotifles/zshrc symlink to ~/.zshrc

VIM setup:
1. install Vundle: 
   git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
2. Run the Vundle install process. Open Vim and execute ':BundleInstall'.
